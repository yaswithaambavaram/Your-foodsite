# 🍔 Your FoodSite

A responsive food ordering website built using HTML, CSS, JavaScript, and React.

---

## 🚀 Features
- Food menu display with images
- Add to cart functionality
- Remove items from cart
- Simple order placement UI
- Fully responsive design (mobile + desktop)

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- React

---

## 🎯 Project Purpose
This project is built to practice front-end development skills and improve understanding of React components, state management, and UI design.
